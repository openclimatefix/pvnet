NMAE cross-comparison

Cross-comparison of models by NMAE is calculated as follows:

value = 100 * (column_model_nmae - row_model_nmae) / column_model_nmae

Which is the percentage improvement (or deterioration if negative) of performance of row model compared to the column model. I.e. each row can be read as "[row model] is better than [column model] by [value]%".
